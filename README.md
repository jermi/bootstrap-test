bootstrap test project + grunt automation
=========================================

Requirements:
-------------
- *[Maven](http://maven.apache.org/ )*
- *[NodeJS + npm](http://nodejs.org/)*

How to run:
-----------
1.  **npm install** 
2.  **npm install -g grunt-cli**
3.  **grunt prod** (for development use '***dev***')
4.  **mvn tomcat:run**
